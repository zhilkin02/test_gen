# **App Name**: ТестГен

## Core Features:

- Загрузка файлов: Загрузка файлов лекций в различных форматах (текст, изображения).
- Анализ текста и изображений: Анализ загруженного текста и изображений с использованием LLM tool для выявления ключевых понятий и тем.
- Генерация вопросов: Генерация вариантов тестовых вопросов на основе выделенных ключевых понятий и тем с использованием LLM tool. Предоставление возможности выбора количества и уровня сложности вопросов.
- Предпросмотр и редактирование: Отображение сгенерированных вопросов с возможностью выбора и редактирования.
- Сохранение вопросов: Возможность сохранения выбранных вопросов в файл.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), evoking trust and intelligence.
- Background color: White, providing a clean and unobtrusive backdrop.
- Accent color: Vibrant red (#F44336), used sparingly for interactive elements to draw attention without overwhelming.
- Clean, sans-serif font for questions and answers, ensuring easy readability.
- Simple, clear icons for file upload, settings, and question types.
- Well-spaced layout with clear visual hierarchy, making it easy to navigate the app and focus on the content.
- Варианты тем интерфейса.
